Crystal Collector

Match the given number by clicking the crystals.

The crystals may look the same but their value changes after each round...

Crystal Collector is a web application game where you click on any of the 4 crystals which have a mysterious value attached. The goal is to match the given number. Beware of the new value the crystals inherit after each round!

Made with HTML/CSS, JavaScript, and jQuery.
